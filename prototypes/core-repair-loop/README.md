# Prototype: Core Repair Loop

**Status**: In-progress — awaiting playtest

## Hypothesis Being Tested

Is the **inspect → replace organ → press Run** loop satisfying as a standalone
mechanic, before any VFX, audio, or narrative is added?

## How to Run

1. Open **Godot 4.6**
2. **Import Project** → select this directory (`prototypes/core-repair-loop/`)
3. Press **F5** (or the ▶ Play button) to run
4. Play on desktop, or export to Android for device testing

The game opens immediately to the puzzle — no menus.

## What to Do

- Tap **Slot 2** (it has the wrong organ — you can tell by the repeated color)
- Tap **Valdris Gate** from the inventory panel below
- Press **▶ RUN SIMULATION**
